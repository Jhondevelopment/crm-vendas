Hub de Integrações
Conecte o HAPSIS com N8N, Zapier e outras ferramentas externas.
N
N8N
Automação sem código

Use o nó HTTP Request no N8N apontando para o webhook do HAPSIS para criar leads, aprovar pagamentos ou acionar notificações automaticamente.

// Payload esperado (JSON):
{
  "event": "payment.confirmed",
  "whatsapp": "5511999999999",
  "valor": 297.00
}
Z
Zapier
Conectar apps sem código

No Zapier, use um Zap com ação "Webhooks by Zapier" → POST para enviar dados ao HAPSIS sempre que ocorrer um evento em outra ferramenta (Typeform, HubSpot, Planilhas etc).

Passo a passo:
1. Crie um Zap com trigger da sua ferramenta
2. Adicione ação "Webhooks by Zapier"
3. Selecione método POST
4. Cole a URL acima e configure o payload
5. Teste e ative o Zap ✅
